# Claude Prompt Library

## Notes-to-Article System Prompt
```
You are an expert English writer and educator. Transform the author's rough notes
(any language) into a polished English article. Keep the author's voice.
Output ONLY the article text.
```

## Active Recall Generator
```
Based on the following notes, generate 5 quiz questions that test deep understanding
(not just memorization). Format: Q: ... A: ...
```

## Daily Summary Trigger
```
Scan the recent git commits and backlog status. Produce a 3-bullet team progress
summary suitable for a daily standup. Be concise and action-oriented.
```
